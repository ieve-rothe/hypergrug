# hypergrug

Hyperdimensional Computing (HDC) library for Crystal Lang.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     hypergrug:
       github: ieve-rothe/hypergrug
   ```

2. Run `shards install`
